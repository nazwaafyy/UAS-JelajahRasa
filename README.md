# Wisata Kuliner App — Platform Promosi & Eksplorasi Kuliner Lokal

Wisata Kuliner App adalah aplikasi mobile berbasis Android yang berfungsi sebagai platform informasi dan promosi kuliner lokal. Aplikasi ini menggabungkan konsep direktori tempat makan dan media sosial, sehingga pengguna dapat menemukan, mengeksplorasi, serta mempromosikan tempat kuliner seperti café, restoran, dan warung dalam satu aplikasi.

Aplikasi ini bertujuan menyediakan informasi kuliner yang lengkap dan terpusat. Pengguna dapat mencari referensi tempat makan berdasarkan lokasi, kategori, harga, dan rating, sementara pemilik usaha dapat mempromosikan produknya melalui konten visual berupa foto dan video.

## Fitur Utama
- Registrasi dan login pengguna
- Feed postingan kuliner
- Profil tempat makan berisi informasi lengkap
- Unggah konten foto dan video
- Pencarian dan filter kuliner
- Integrasi Google Maps
- Bookmark dan favorit
- Review dan rating (opsional)

## Teknologi yang Digunakan
- Flutter
- PHP dan MySQL
- Google Maps API

## Tujuan Pengembangan dan Peran Pengguna
Aplikasi Wisata Kuliner App dikembangkan sebagai pusat informasi kuliner yang mudah diakses sekaligus sebagai sarana promosi digital bagi UMKM kuliner lokal. Aplikasi ini membantu pengguna umum atau wisatawan dalam mencari, melihat, dan menyimpan referensi tempat makan, serta membantu pemilik usaha dalam mengelola profil bisnis dan mempromosikan produk kuliner mereka melalui konten visual. Dengan adanya aplikasi ini, diharapkan dapat meningkatkan daya tarik wisata kuliner daerah dan mendukung digitalisasi sektor kuliner lokal.

## Cara Menjalankan Aplikasi
Aplikasi dijalankan menggunakan Flutter dengan memastikan Flutter SDK telah terpasang dan terkonfigurasi dengan baik, yang dapat dicek melalui perintah flutter doctor. Setelah membuka folder proyek Flutter, dependency aplikasi diunduh menggunakan flutter pub get, kemudian aplikasi dijalankan menggunakan flutter run pada emulator, browser, atau perangkat fisik. Agar seluruh fitur berjalan dengan baik, backend berbasis PHP dan MySQL harus dijalankan melalui server lokal seperti XAMPP, serta alamat API pada aplikasi disesuaikan dengan alamat server yang digunakan.
